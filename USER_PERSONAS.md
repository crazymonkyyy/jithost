# Demo User Personas and Their Website Specifications

## User 1: Alex Chen - Tech Blogger
**Personality**: Alex is a 28-year-old software engineer who loves to write technical tutorials and reviews. Detail-oriented, loves clean design, and wants to focus on code snippets and technical diagrams. Alex is pragmatic, tech-savvy, and prefers minimalist aesthetics.

### Alex's Website SPEC.md
```
# Alex's Tech Blog

## Purpose
Share technical tutorials, code reviews, and programming tips with the developer community.

## Content Requirements
- Code highlighting with multiple language support
- Inline diagrams using mermaid syntax
- Technical deep-dives with collapsible sections
- Tutorial series with navigation between parts

## Design Preferences
- Dark theme as default (for coding late nights)
- Clean, minimalist layout
- Wide code blocks (no horizontal scroll)
- Font optimized for code readability

## Technical Needs
- Syntax highlighting for: D, Rust, Python, JavaScript, Go
- Math equation support (LaTeX)
- Integration with GitHub gists
- Comments section (but off by default to reduce noise)
- RSS feed for subscribers

## Content Structure
- Blog posts (chronological)
- Tutorials (organized by topic/level)
- Code snippets library
- Book reviews
- Tools I use page
```

## User 2: Maya Patel - Food Photographer & Recipe Creator
**Personality**: Maya is a 32-year-old food photographer and home cook with an eye for aesthetics. She's creative, passionate about visual storytelling, and wants to showcase her photography. Maya's warm and inviting personality should come through her website.

### Maya's Website SPEC.md
```
# Maya's Culinary Canvas

## Purpose
Showcase my food photography and share recipes that tell stories and bring people together.

## Content Requirements
- High-quality image gallery for food photography
- Recipe format with ingredients, steps, and cook time
- Seasonal content highlighting fresh ingredients
- Step-by-step photo tutorials
- Stories behind recipes (family traditions, etc.)

## Design Preferences
- Warm, inviting color palette
- Large, beautiful photos as primary visual elements
- Easy-to-read recipe format
- Seasonal themes matching food trends
- Mobile-optimized for recipe viewing

## Technical Needs
- Responsive image gallery
- Print-friendly recipe format
- Social sharing integration
- Search by ingredients or dietary restrictions
- Email newsletter signup
- SEO optimized for food searches

## Content Structure
- Recipe archives by category (appetizers, mains, desserts, etc.)
- Seasonal highlights
- Photography portfolio
- About page with personal story
- Contact and collaboration info
```

## User 3: David Rodriguez - Indie Game Dev & Designer
**Personality**: David is a 26-year-old indie game developer who loves retro aesthetics, pixel art, and game mechanics. He's creative, nostalgic, and wants to showcase interactive elements. David appreciates quirky design and enjoys engaging with his audience.

```
# David's Game Lab

## Purpose
Portfolio of my indie games, development process, and experimental projects. A place to share the joy of game creation.

## Content Requirements
- Game demos playable directly in browser
- Development blog with technical details
- Interactive elements (demos, mini-games)
- Pixel art showcase
- Game jam entries and prototypes

## Design Preferences
- Retro gaming aesthetic (8-bit/16-bit inspired)
- Animated elements and easter eggs
- Game-like navigation (menu screens, etc.)
- Nostalgic color palettes
- Custom cursors and UI elements

- Embedded web-based game demos
- Canvas/JavaScript for interactive elements
- Audio support for game sound samples

- Patreon integration
- Game download links

## Content Structure
- Game portfolio with playable demos
- Development blog (post-mortems, tutorials)
- Pixel art gallery
- Game design resources
```

## User 4: Elena Volkov - Environmental Scientist & Consultant
**Personality**: Elena is a 35-year-old environmental scientist who is passionate about data visualization and environmental education. She's analytical, evidence-based, and wants to make complex topics accessible to a broader audience. Elena values professionalism and accuracy.

### Elena's Website SPEC.md
```
# Elena's Environmental Insights

## Purpose
Share research findings, environmental data, and consulting services to promote sustainable practices and climate awareness.

## Content Requirements
- Data visualization and interactive charts
- Research papers and summaries
- Environmental impact calculators
- Educational resources and infographics
- Consulting services and case studies

## Design Preferences
- Professional and clean layout
- Nature-inspired color palette (greens, blues, earth tones)
- Data-first focus with clear visualizations
- Easy-to-scan content for research
- Accessibility compliance

## Technical Needs
- Interactive data visualizations (D3.js or similar)
- PDF generation for reports
- Form integration for consultation requests
- Multi-language support (English/Spanish/French)
- Academic citation formatting
- Subscription to research updates

## Content Structure
- Research publications and summaries
- Environmental resources and guides
- Consulting services and case studies
- Speaking engagements and events
- About and professional background
- Contact and consultation request form
```
