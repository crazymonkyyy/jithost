# Introduction to JitHost

Welcome to JitHost, a toolkit HTML server that makes it easy to do everything from exploring a folder of markdown documents to publishing a website with exact HTML.

## What is JitHost?

JitHost is a static site generator with JIT hosting capabilities. This means you can either:
- Pre-build all pages (static generation)
- Generate pages on-demand (JIT hosting)

## Getting Started

To get started with JitHost, simply create a markdown file and serve it!
