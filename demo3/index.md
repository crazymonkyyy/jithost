## About This Demo

This demo shows how JitHost can be used to create a complex site structure with multiple levels of hierarchy, with consistent navigation and layout across all pages.

```html <class=sidebar>
<div class="widget">
  <h3>Site Navigation</h3>
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/posts/index.html">All Posts</a></li>
    <li><a href="/categories/index.html">Categories</a></li>
    <li><a href="/tags/index.html">Tags</a></li>
  </ul>
</div>
<div class="widget">
  <h3>Archives</h3>
  <ul>
    <li><a href="/archive/2023/index.html">2023</a></li>
    <li><a href="/archive/2022/index.html">2022</a></li>
  </ul>
</div>
```

### Key Features Demonstrated

- Multi-level site hierarchy
- Complex navigation structure
- Consistent styling across all pages
- HTML embedding for custom components
- Support for nested directory structures
